# Project Brief

Yerel bilgisayarda calisan Python tabanli bir e-Arsiv taslak fatura otomasyonu gelistiriliyor.

V1 hedefi, kullanicinin Excel/CSV kaynaktan kayitlari import edip secmesi, Playwright ile portala girerek her secili kisi icin taslak e-Arsiv faturasi olusturmasi ve hata veren kisileri batch'i durdurmadan raporlamasidir.

Faz 1 kapsaminda sadece calisan proje iskeleti, SQLite veri katmani, import servisi ve lokal web panelde kayit listeleme uygulanir.
